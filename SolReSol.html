SolReSol

SolReSol was inspired by Clark'sSuperscope video. Using MAX/MSP anda tool to convert SVG files to stereo audio output, I created an interface for a performer to mix audio channels through a visual output, allowing them to "DJ" various lissajous patterns. Unfortuntaely, this relies on rather expensive proprietary software to run. In the future I would like to port it to the FOSSPure Data.
